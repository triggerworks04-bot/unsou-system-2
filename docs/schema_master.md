# スキーママスタ（列定義の元本）

**本ファイルが列定義の正本です。**  
新しい列の追加・型・意味の変更は、**必ず本ファイルを先に更新**し、その後スプレッドシート／AppSheet／GAS に反映してください。

## 共通ルール

- 論理名（日本語）と物理名（推奨英字スネークケースまたはシート事情に合わせた列名）を対応させる。
- **運行予定ID** は配車予定表を起点とし、他テーブルから参照できるようにする。
- **運転者ID・車両ID・案件ID・取引先ID** は共通マスタのキーと一致させる。
- 「単独利用OK」の列は、連携しない運用でも NULL 可または任意入力可とすることを許容する設計とする。
